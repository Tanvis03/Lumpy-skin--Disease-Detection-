# Lumpy Skin Disease Detection

## Project Overview
This project uses deep learning techniques to detect Lumpy Skin Disease (LSD) in cattle from images. The model classifies images as healthy or infected, enabling early disease detection and supporting better livestock health management.

## Features
- Detects Lumpy Skin Disease from cattle images
- Image classification using deep learning
- Early disease detection
- Supports livestock health monitoring

## Technologies Used
- Python
- TensorFlow/Keras
- OpenCV
- NumPy
- Matplotlib

## Future Enhancements
- Improve detection accuracy
- Real-time disease detection
- Web and mobile application deployment
